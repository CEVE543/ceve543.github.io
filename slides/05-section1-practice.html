Bayes Rule

A doctor is called to see a sick child. The doctor has prior information that 90% of sick children in that neighborhood have the flu, while the other 10% are sick with measles. Assume for simplicity that there are no other illnesses and that no children have both the flu and measles.

A well-known symptom of measles is a rash. The probability of having a rash if one has measles is 0.95. However, occasionally children with flu also develop rash, and the probability of having a rash if one has flu is 0.08. Upon examining the child, the doctor finds a rash. What is the probability that the child has measles?

Defing the events

  • Let \(M\) be the event that the child has measles
  • Let \(F\) be the event that the child has the flu
  • Let \(R\) be the event that the child has a rash
  • We want \(P(M | R)\)

Note: we are implicitly conditioning on the nieghborhood and the kid being sick!

Bayes’ Rule

\[ P(M | R) = \frac{P(R | M) P(M)}{P(R)} \]

  • \(P(M) = 0.1\)
  • \(P(F) = 0.9\)
  • \(P(R | M) = 0.95\)
  • \(P(R | F) = 0.08\)
  • \(P(R) = P(R | M) P(M) + P(R | F) P(F)\)

Calculation

PR_measles = 0.1
PR_rash_given_measles = 0.95
PR_rash_given_flu = 0.08
PR_flu = 1 - PR_measles
PR_rash = PR_rash_given_measles * PR_measles + PR_rash_given_flu * PR_flu

PR_measles_given_rash = PR_rash_given_measles * PR_measles / PR_rash

Analytic posterior

We collect some count data and model it using a Poisson likelihood. The Poisson likelihood is given by: \[ p(y_i | \lambda) = \frac{\lambda^{y_i} e^{-\lambda}}{y_i!} \] where \(y_i\) is the number of counts and \(\lambda\) is the rate parameter of the Poisson distribution. We want to do inference on \(\lambda\). We have a prior belief that \(\lambda\) is distributed as an Exponential distribution with the modified parameterization: \[ p(\lambda | \theta) = \frac{1}{\theta} e^{-\frac{\lambda}{\theta}} \] After collecting data \(y_1, y_2, \ldots, y_n\), what is the posterior distribution of \(\lambda\) given our prior parameter \(\theta\)?

Getting oriented

We want the posterior: \[ p(\lambda | y) \propto p(\lambda) p(y | \lambda) \] it will be convenient to work with the log of the posterior: \[ \log p(\lambda | y) = \log p(\lambda) + \log p(y | \lambda) + C_1 \] where we don’t care about \(C_1\). Let’s work through step by step.

Log likelihood

For a single data point, we have \[ p(y_i | \lambda) = \frac{\lambda^{y_i} e^{-\lambda}}{y_i!} \] and so for a single data point we have \[ \log p(y_i | \lambda) = \log \left( \frac{\lambda^{y_i} e^{-\lambda}}{y_i!} \right) = y_i \log(\lambda) - \lambda - \log(y_i!) \] For multiple data points, we have \[ \log p(y | \lambda) = \sum_{i=1}^{n} \left( y_i \log(\lambda) - \lambda - \log(y_i!) \right) \]

Log posterior

For the log prior with the modified parameterization, we have \[ \log p(\lambda | \theta) = \log \left( \frac{1}{\theta} e^{-\frac{\lambda}{\theta}} \right) = -\log(\theta) - \frac{\lambda}{\theta} \] Putting it all together, we have \[ \log p(\lambda | y) = -\log(\theta) - \frac{\lambda}{\theta} + \sum_{i=1}^{n} \left( y_i \log(\lambda) - \lambda - \log(y_i!) \right) + C_2 \] where ( C_2 ) is a constant term.

Checking

We can implement this function in Julia:

function log_posterior(λ::T, y::Int, θ::T) where {T<:Real}
    # Calculate the log prior with the modified parameterization
    log_prior = -log(θ) - (λ / θ)

    # Calculate the log likelihood for a single data point y
    log_likelihood = y * log(λ) - λ - log(factorial(y))

    # Return the sum of log prior and log likelihood
    return log_prior + log_likelihood
end

function log_posterior(λ::T, y::AbstractVector, θ::T) where {T<:Real}
    return sum(log_posterior.(λ, y, θ))
end

function log_posterior_2(λ::T, y::Int, θ::T) where {T<:Real}
    log_prior = logpdf(Exponential(θ), λ)
    log_likelihood = logpdf(Poisson(λ), y)
    return log_prior + log_likelihood
end

function log_posterior2(λ::T, y::AbstractVector, θ::T) where {T<:Real}
    return sum(log_posterior_2.(λ, y, θ))
end
